## 2.
---
`search_hosts.ini`について
```
[hosts]
# 検索対象とするホスト名（FQDN）を記載します
# host1 = host1.local
# serverA = serverA.local
# printer1 = printer1.local
test-device.local = test-device.local
```

例えば次のように書いても動作するようにしてほしい。
```
[hosts]
# 検索対象とするローカルホスト名またはローカルFQDNを記載します
# host1
# serverA
# printer1.local
test-device1
test-device2.local
```
※この場合、`test-device1.local`と`test-device2.local`が対象になる。

## 4.
---
@/5_ソースコード\mdns_proxy  HTTP待ち受けTCPポートを変更できるようにコードとiniに機能追加して。
- 初期値は80で良い。
- `system.ini`は次の様な感じになりますかね。
```
[system]
# 実行間隔（秒）
interval = 30
# 発信トークン接頭語
token_prefix = mDNSProxy_
# HTTP待ち受けポート
port = 80
# TTL初期値（秒）
ttl = 120

[network]
# 外部mDNSプロキシIPアドレスとポート（カンマ区切りで複数指定可能）
external_proxies = 192.168.1.10:80,192.168.2.10:8080

# Wi-Fi設定（Pico用）
wifi_ssid = your_wifi_ssid
wifi_password = your_wifi_password
```
